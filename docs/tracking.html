Progreso y feedback

Seguimiento simple, sin secretos.

La meta es poder saber qué avanzó cada estudiante, qué entendió y qué necesita otra explicación. Todo con un flujo liviano y transparente.

Seguimiento + feedback

El avance no se lee solo por módulos completados. También importa lo que cada estudiante reporta después de la clase: dudas, claridad, ritmo y confianza.

Ir a Feedback Guía para mentores

Flujo conceptual

A

Entrada del estudiante

La reflexión breve de Feedback captura avance, dudas y señales de comprensión.

B

Registro central

Una tabla común ordena respuestas y métricas sin exponer datos privados.

C

Lectura de progreso

Docentes o mentores ven patrones: qué se entiende, qué se repite y qué falta.

D

Mejora del contenido

Con esos datos se ajustan explicaciones, ejemplos y orden de la ruta.

Qué conviene guardar

  • Progreso por módulo.
  • Dudas frecuentes.
  • Autoevaluación breve del estudiante.
  • Comentarios sobre claridad, ritmo y dificultad.

Modelo mínimo de datos

Para empezar con Google Sheets o AppSheet, alcanza con una tabla simple. Podés usar este CSV de ejemplo como punto de partida.

  • timestamp: cuándo se registró el feedback.
  • student_alias: nombre corto o alias del estudiante.
  • module: módulo o clase asociada.
  • confidence_level: bajo, medio o alto.
  • understood: qué entendió.
  • blocked_on: dónde se trabó.
  • needs_next: qué necesita para avanzar.
  • evidence_link: evidencia opcional, si existe.
  • status: started, practiced, feedback_sent, needs_support o completed.
  • mentor_notes: observaciones internas del mentor.

Si vas a usar AppSheet

La configuración pensada para mentores y administración está documentada en docs/appsheet-setup.html. Esa guía explica cómo pasar del formulario a la hoja y de la hoja a las vistas de seguimiento.

Qué no conviene guardar

  • Credenciales o secretos.
  • Enlaces privados dentro del repo público.
  • Datos sensibles que no aporten al aprendizaje.

Nota sobre analítica/tracking

Si el repositorio ya tiene una configuración de analítica o medición, debe mantenerse. En este slice no se detectó una configuración activa, así que el tracking sigue siendo conceptual y se apoya en la página de Feedback para recolectar señales de aprendizaje. Cuando exista el formulario real, el link se puede agregar sin publicar datos privados.
