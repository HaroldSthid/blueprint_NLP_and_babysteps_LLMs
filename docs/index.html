Qué es este proyecto

Una capa pública y simple sobre el notebook original para que cualquier estudiante pueda entender qué estudiar, en qué orden y cómo pedir ayuda.

Para quién es

Personas sin base técnica fuerte que quieren entrar a NLP y LLMs sin fricción.

Cómo se usa

Leé la ruta en GitHub cuando necesites revisar; usá Colab para ejecutar y practicar el notebook guiado.

Qué se mide

Avance, feedback y señales de comprensión, sin exponer datos privados.
