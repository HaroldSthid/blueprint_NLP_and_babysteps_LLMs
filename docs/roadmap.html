Fases sugeridas para Track 1

Fase Objetivo Señal de avance
1. Track 1: Módulo 1 Entender cómo una máquina lee texto antes de entrar al notebook. Podés explicar la idea base con tus palabras.
2. Orientación Entender de qué trata NLP y qué problema resuelve. Podés explicarlo con tus palabras.
3. Primer contacto Ejecutar el notebook y reconocer cada bloque. Sabés dónde mirar si algo no se entiende.
4. Práctica guiada Modificar inputs, observar salidas y comparar resultados. Podés repetir el ejercicio sin ayuda directa.
5. Lectura crítica Preguntarte qué hace el modelo y qué no hace. Detectás límites y posibles mejoras.

Buenas prácticas para estudiantes

  • No saltes de tema antes de entender el anterior.
  • Escribí dudas concretas, no solo “no entendí nada”.
  • Repetí ejercicios con pequeños cambios.
  • Usá el roadmap como checklist, no como examen.
