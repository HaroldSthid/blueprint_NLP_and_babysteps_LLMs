Ruta de aprendizaje

Una secuencia simple para no perderte.

La idea no es memorizar todo de golpe. Es construir confianza: primero leer, después practicar, y recién ahí profundizar.

Fases sugeridas

Fase Objetivo Señal de avance
1. Orientación Entender de qué trata NLP y qué problema resuelve. Podés explicarlo con tus palabras.
2. Primer contacto Ejecutar el notebook y reconocer cada bloque. Sabés dónde mirar si algo no se entiende.
3. Práctica guiada Modificar inputs, observar salidas y comparar resultados. Podés repetir el ejercicio sin ayuda directa.
4. Lectura crítica Preguntarte qué hace el modelo y qué no hace. Detectás límites y posibles mejoras.

Qué conviene mirar primero

  • La página principal para entender el contexto general.
  • El notebook para ver el contenido real del curso.
  • Seguimiento para saber cómo se registra feedback y progreso.

Buenas prácticas para estudiantes

  • No saltes de tema antes de entender el anterior.
  • Escribí dudas concretas, no solo “no entendí nada”.
  • Repetí ejercicios con pequeños cambios.
  • Usá el roadmap como checklist, no como examen.
